# Search Sets

SearchSets are a powerful tool for filtering, organizing, and reusing searches across the construction model data.

### Use of SearchSets

Filter and group data: You can create custom groups of issues, objects, or elements based on properties like type, status, geometry, or custom parameters.

Quick access: Instead of manually searching every time, you save a SearchSet to instantly retrieve the same group of items.

Collaboration: Teams can share SearchSets to ensure everyone is looking at the same filtered view.

### How SearchSets Work

#### 1. Create a search:

Define filters (e.g., all items with the same property value, or all doors with a specific type).

#### 2. Save it as a SearchSet:

Give it a name and optional color tag for easy identification.

#### 3. Apply SearchSet anywhere:

In issues, 3D models, or 2D sheets, you can apply the SearchSet to instantly see only the filtered items.

#### 4. Dynamic updates:

If new items match the SearchSet criteria, they automatically appear in it without needing to recreate the search.
